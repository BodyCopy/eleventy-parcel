A basic Eleventy (11ty) starter project with Parcel JS.

## Features

- Nunjucks
- Parcel for JS bundling and Sass compilation
- Basic Sass architecture to get you started
- Two initial layout templates

## Commands

### Run

```
npm start
```

### Build for production

```
npm run build
```

### Generate SVG sprite

```
npm run svg
```
